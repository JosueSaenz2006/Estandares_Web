Tipos de etiquetas en html usadas en esta página web

head:

Generalmente contiene información que necesitamos que se cargue antes de iniciar la página.


nav:

Proporciona enlaces de navegación para mejorar la experiencia de usuario.


section:

Representa una sección dentro de la página, diferenciandola de otras. Puede agrupar contenido relacionado como un conjunto de artículos.


article:

Puede almacenar un contenido el cual puede ser reutilizado o distribuido.


footer:

Es una etiqueta que define el pie de página de un sitio web o de una sección de contenido.


aside:

Con él se puede agregar información adicional al contenido principal. El contenido que se use dentro de está etiqueta se puede considerar como no esencial para el usuario
